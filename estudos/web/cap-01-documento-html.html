1 — Documento HTML e o index.html

Um arquivo .html é texto com marcação: o navegador constrói uma árvore (DOM). O <head> carrega metadados e recursos; o <body> é o que o usuário vê.

O que abrir no repositório

Comece por index.html na raiz. Note <main id="main-content"> (conteúdo principal), links para css/tokens.css, css/style.css e scripts com defer.

O link Pular para o conteúdo melhora acessibilidade: mesmo padrão está nas páginas de estudo.
